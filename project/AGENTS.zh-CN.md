## 名称说明

本文档是 **项目级 Agent 工作规范** 模板。这里的“项目级 Agent 工作规范”指当前仓库专属的 AI 编程助手工作约定集合，包括技术栈、常用命令、目录结构、项目约定、验证要求、风险区域和项目经验。

具体文件名可能因工具而异，例如 `AGENTS.md`、`CLAUDE.md`、`GEMINI.md`，或某个 wrapper/import 文件。正文使用“项目级 Agent 工作规范”，而不是写死 `AGENTS.md`，是为了让同一份内容被不同工具复用时仍然语义清晰。
